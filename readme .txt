Documentul este in format HTML(index.html) si se va deschide direct in browser pentru a vizualiza datele.

Decizi tehnice
- am adaugat 2 coloane in html, una pentru afisarea profilului de user si a doua coloana pentru lista userilor.
- am setat constantele unde vom afisa datele din ambele coloane - col A "sidebar"(datele de profil) si coloana B "main"(lista tuturor userilor).
- am construit functie pentru preluarea userilor existenti - "getUsers()" si si o alta pentru preluarea datelor pentru fiecare user in parte - "getUser(username)" .
- am construit o functie pentru a afisha lista userilor "showUsers(usersPromise)" cu ajutorul datelor preluate din getUsers().
- in final am adaugat o functie pentru a afisha informatile fiecarui user existent - "showProfile(user)" atunci cand va fi apelat prin apasarea butonului "Details" afisand informatile existente in sidebar pentru userul apelat.
